# UNP_PROJECT_E_COMMERCE_DATA_ANALYSIS_USING_MySQL

I worked on this project in collaboration with UNP, where I analyzed e-commerce data using MySQL to extract meaningful business insights. The dataset included customers, orders, products, and sales details, allowing a complete view of business operations.

Using SQL queries like SELECT, JOIN, GROUP BY, and aggregate functions, I analyzed sales performance, customer behavior, and product trends. I identified key metrics such as total sales, average orders per customer, and order patterns across different time periods.

The project helped in identifying top-selling products, high-value customers, and overall revenue trends. I also explored customer engagement by analyzing customers with and without orders, along with payment and order details.

Overall, this project strengthened my SQL and data analysis skills and demonstrated how structured data can be used to support better business decisions and improve performance.
